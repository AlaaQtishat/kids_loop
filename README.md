## 📖 About The Project
Children grow up fast, and their clothes often become too small long before they are worn out. **KidsLoop** is a mobile application developed during my training at **dot jo**. It connects local parents to buy and sell pre-loved children's items, providing an economical and sustainable solution to the "fast-growing kids" challenge.

## ✨ Key Features
* **🛒 Product Management:** Easily add, edit, or delete items and manage their status.
* **💬 Real-time Chat:** Seamless communication between buyers and sellers.
* **❤️ Smart Favorites:** Save items with high performance using **Optimistic UI** updates.
* **🌗 Dark & Light Mode:** A visually comfortable interface that supports both themes.
* **🌍 Multi-language Support:** Fully localized in both Arabic and English.
* **🔔 Push Notifications:** Instant alerts for new messages using OneSignal.
* **🛡️ Secure Authentication:** Reliable login and signup system powered by Firebase Auth.

## 🛠️ Tech Stack

### Frontend
* **Framework:** [Flutter](https://flutter.dev/) (Dart)
* **State Management:** Provider
* **Localization:** easy_localization
* **Local Storage:** SharedPreferences

### Backend & Services
* **Database & Auth:** Firebase (Firestore & Authentication)
* **Notifications:** OneSignal
* **Design Tools:** Figma
